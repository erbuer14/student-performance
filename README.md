# Student Performance SQL Analysis

## Overview
This project analyzes factors that influence student exam performance using SQL. 
The dataset includes variables related to study habits, attendance, sleep, tutoring, and extracurricular participation.

## Dataset
The dataset is located in:
- `data/student_performance.csv`

It contains anonymized student-level data used for SQL analysis.

## Questions Answered
1. Do higher study hours and extracurricular activities lead to better scores?
2. Is there a sweet spot for study hours?
3. How can students be ranked without revealing exam scores?

## Repository Structure
- `analysis.sql` – SQL queries used to answer the business questions  
- `student_performance.csv` – Dataset used for analysis  
- `exploration.ipynb` – Exploratory analysis notebook


## Tools Used
- SQL
